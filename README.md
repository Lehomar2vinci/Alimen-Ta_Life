# Alimen'Ta Life

Site pour promouvoir **Alimen'Ta Life : Le Réseau des GASAP**.

## Aperçu

Le site comprend :

- une page d’accueil très animée ;
- une direction graphique pop, colorée et fantaisiste ;
- une boule alimentaire roulante avec objets collés ;
- une section explicative sur les GASAP ;
- une section fonctionnement ;
- une section impact ;
- des témoignages humoristiques de légumes ;
- une section “galaxie de paniers” ;
- une section de soutien avec lien Streamlabs Charity ;
- des animations CSS et JavaScript ;
- un mode “Secouer la salade” ;
- un mode “Orbite cosmique” ;
- une jauge de taille de boule ;
- des confettis, sons synthétiques et pluie de légumes.
